# 修仙宗门模拟器 🎮

一个基于网页的修仙宗门管理模拟游戏，采用鬼谷八荒风格。

## 🎯 游戏特色

- **宗门管理**：招募弟子、管理资源、发展宗门
- **弟子系统**：详细的弟子属性、天赋词条、好感度系统
- **修炼体系**：完整的修为境界、突破系统
- **事件系统**：随机事件、集体挑战
- **AI交互**：智能弟子对话系统

## 🚀 快速开始

### 在线游玩
访问：https://sunsetzf2023.github.io/text-simulator-beta/

### 本地运行
```bash
# 使用 Python（推荐）
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- **前端**：原生 JavaScript (ES6 Modules)
- **样式**：Tailwind CSS
- **部署**：GitHub Pages

## 📦 项目结构

```
生化斯坦/
├── index.html          # 主页面
├── src/                # 源代码
│   ├── main.js         # 游戏主逻辑
│   ├── state.js        # 游戏状态管理
│   ├── models/         # 数据模型
│   ├── utils/          # 工具函数
│   ├── ai/             # AI系统
│   └── data/           # 游戏数据
├── style.css           # 样式文件
└── README.md           # 项目说明
```

## 🎮 游戏操作

1. **开始游戏**：输入宗门名和玩家名
2. **资源管理**：采集灵石、突破境界
3. **弟子管理**：招募弟子、查看详情、改名培养
4. **宗门发展**：任务堂、坊市、拍卖会
5. **事件应对**：处理随机事件和挑战

## 🔧 开发说明

### ES6 模块
项目使用 ES6 模块系统，需要现代浏览器支持。

### 本地开发
由于 CORS 限制，本地开发需要使用 HTTP 服务器，不能直接打开 HTML 文件。
